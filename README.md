BASTA - web

Lightweight React-based local web implementation of the Mexican word game "BASTA" (also known as Tutti Frutti / Scattergories).

How to run:
- Serve the directory with a static server, e.g. `python3 -m http.server` and open http://localhost:8000

Notes:
- Uses ESM React from CDN so no npm install is required.
- Simple scoring: unique=10, duplicate=5, blank=0.
